description: NUNCHI 프로젝트의 Git 커밋 메시지, 브랜치명, 이슈 제목, PR 제목 작성 규칙
type: feedback
---

커밋 메시지, 브랜치명, 이슈 제목, PR 제목은 아래 포맷을 따른다.

- **커밋**: `[Feat] 회원가입 기능 구현`
- **브랜치**: `feat/#1/sign-up` (타입/이슈번호/기능명-kebab-case)
- **이슈 제목**: `[Feat] 회원가입 기능 구현`
- **PR 제목**: `[Feat] 회원가입 기능 구현`

**Why:** 팀 그라운드룰로 정해진 컨벤션. 일관성 유지 필수.

**How to apply:** 커밋/브랜치 생성/PR 작성 시 항상 이 포맷 사용. 타입 예시: `[Feat]`, `[Fix]`, `[Refactor]`, `[Docs]`, `[Chore]` 등.
