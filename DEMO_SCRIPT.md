# 🎬 NUNCHI 아바타 모드 — 시연 영상 대본

> 실제 앱(localhost:8080 + 라이브 FastAPI)을 직접 구동하고 Playwright로 흐름을 검증해 작성한 대본입니다.
> **모든 메뉴·가격·옵션은 실제 DB에 존재하는 값**으로만 구성했습니다.
> 목표 분량: 시나리오당 약 1분 20초 ~ 1분 30초.

---

## ⚠️ 촬영 전 반드시 확인 (실측 기반 주의사항)

직접 돌려보며 확인한, 데모에서 사고 나기 쉬운 지점들입니다.

| 구분 | 내용 |
|------|------|
| ✅ **인기 메뉴** | 오늘자 판매 데이터가 반영되어 **"인기 메뉴 / 베스트 추천해줘"** 정상 동작. 오늘의 순위: **1위 일식카레덮밥(50) · 2위 데리야끼치킨솥밥(45) · 3위 철판치즈돈가스(40) · 4위 가츠동(35) · 5위 삼겹살김치철판(32)**. → 시나리오 5 |
| ✅ **추천 유도법** | "인기 메뉴" 외에도 **"안 매운 거 / 담백한 거 / 든든한 거"** 같은 **속성 기반** 추천 모두 가능. |
| ⚠️ **AI 서버 안정성** | 라이브 FastAPI(dev)가 간헐적으로 스트림이 끊기거나(`ERR_INCOMPLETE_CHUNKED_ENCODING`) 세션 시작이 실패함. **촬영 전 워밍업 + 세션 1회 선실행**으로 안정화 후 찍을 것. |
| 💡 **가장 안전한 흐름** | 추천에 의존하지 말고 **메뉴 이름을 직접 말해서 담기 → 옵션 → 결제**가 제일 확실함. (시나리오 1·2) |
| 🔁 **필수 옵션** | **국 선택·소스 선택은 필수 옵션**이라 AI가 반드시 되묻음 → "되묻기→답변" 장면이 자연스럽게 연출됨. |

### 진입 경로 (검증 완료)
```
시작화면 [AI 대화 주문]  →  /mode  →  /dine [매장에서]  →  /avatar (눈치 등장 + 음성 인사)
```
진입 직후 실제 인사 대사:
> **눈치:** "안녕하세요! 무엇을 도와드릴까요? 메뉴를 추천해드릴까요, 아니면 직접 골라보시겠어요?"

## 📌 강조하면 좋은 화면 요소 (촬영 포인트)

- 마이크 버튼의 **ripple(물결) 애니메이션** — "듣고 있다"는 인상
- 말풍선 **타이핑 효과 ↔ 음성(TTS) 동기** — 아바타가 살아있는 느낌
- **옵션 시트 / 추천 시트 슬라이드업**
- 미니카트에 메뉴가 **담기는 순간**과 합계 변화
- 상단 **단계 인디케이터**가 채워지는 모습 (기→승→전→결)

# 🎬 시나리오 1 — "이름만 말하면 끝" (가장 안전 · 권장)

> **콘셉트:** 추천 기능에 의존하지 않는 가장 확실한 흐름. 음성 자율주문의 속도·편의성 강조.
> **러닝타임:** 약 80초 · **실제 합계 9,000원**

| # | 시간 | 화면 / 연출 | 화자 | 대사 |
|---|------|------------|------|------|
| 1 | 0:00–0:10 | 시작화면→[AI 대화 주문]→[매장에서] 터치 | 자막 | "말로 주문하는 키오스크, 눈치" |
| 2 | 0:10–0:16 | 아바타 등장 · 손 흔들며 인사 | 🤖 눈치 | "안녕하세요! 무엇을 도와드릴까요? 메뉴를 추천해드릴까요, 아니면 직접 골라보시겠어요?" |
| 3 | 0:16–0:24 | 마이크 버튼 누름(물결 애니메이션) | 🙋 사용자 | "데리야끼치킨솥밥 하나 주세요." |
| 4 | 0:24–0:30 | 옵션 시트 슬라이드업 (국 선택) | 🤖 눈치 | "데리야끼치킨솥밥이요! 국은 된장국과 미역국 중에 어떤 걸로 드릴까요?" |
| 5 | 0:30–0:36 | 사용자 응답 | 🙋 사용자 | "된장국이요." |
| 6 | 0:36–0:46 | 미니카트에 담김 (₩7,500) | 🤖 눈치 | "된장국으로 담아드렸어요. 더 필요하신 거 있으세요?" |
| 7 | 0:46–0:52 | 음료 추가 발화 | 🙋 사용자 | "콜라 하나 추가해주세요." |
| 8 | 0:52–0:58 | 카트 합계 ₩9,000 갱신 | 🤖 눈치 | "콜라 담았어요! 총 9,000원이에요. 결제 도와드릴까요?" |
| 9 | 0:58–1:04 | 결제 의사 확인 | 🙋 사용자 | "네, 결제할게요." |
| 10 | 1:04–1:20 | 주문요약→결제→완료 화면 | 🤖 눈치 | "주문 완료됐어요. 맛있게 드세요!" |
| — | 1:20 | 마무리 자막 | 자막 | "눈치 — 말로 하는 자율주문" |

**주문 내역:** 데리야끼치킨솥밥(된장국) 7,500 + 콜라 1,500 = **9,000원**

## ▶️ 로컬 실행 메모

```bash
# 인프라 (postgres + redis)
docker compose up -d postgres redis

# Spring (local 프로필 — DataInitializer가 메뉴 시드)
SPRING_PROFILES_ACTIVE=local ./gradlew bootRun

# 접속: http://localhost:8080
# 종료: docker compose down
```

> 라이브 AI(FastAPI)는 `.env`의 `LOCAL_FASTAPI_URL`(sslip.io)에 연결됩니다.
> 이 FastAPI는 배포된 dev Spring DB와 통신하므로, 촬영 환경과 데이터가 일치하는지 사전 확인 필요.
